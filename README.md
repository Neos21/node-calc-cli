# node-calc-cli : @neos21/calc-cli

[![NPM Version](https://img.shields.io/npm/v/@neos21/calc-cli.svg)](https://www.npmjs.com/package/@neos21/calc-cli)

CLI Calculator.


## How To Use

```sh
$ npm install -g @neos21/calc-cli

$ calc '80.7 - 10.1'
# --> 70.6
```
